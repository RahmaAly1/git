# Git Commands

### to initialize git go to your project folder and run 
```
git init

```

### to add all my files to stage run
```
git add .
```

### to commit your code run
```
git commit -m "my message"

```

### to push to your emote branch run
```
git push
```

### to clone a repo run
```
git clone https://github.com/anyproject
```
